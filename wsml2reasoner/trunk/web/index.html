WSML 2 Reasoner Framework

The WSML 2 Reasoner framework can translate ontology descriptions in WSML to predicates and rules. Additionally it provides a facade for easy integration of different reasoners.

Source and binary distributions can be obtained from the Snap Shot directory (generated daily). Please note that this is a work in progress and should at best be considered alpha quality code. We are continually trying to improve the quality of the software. Comments and suggestions are welcome.

Documentation
  • You may browse the source code
  • Or take a look at the Java Doc reports
  • This simple example might help to understand how to interface with the existing code
  • In addition to the example, a brief tutorial provides a list of required libraries and a short explanation of the essential parts of the framework
Demonstration

A simple Online Demo has been developed based up on this framework.

Dependencies

This software depends on WSMO4j (release 0.5.2)

Additionally you will need a reasoning engine. The wrappers are currently implemented and tested for KAON2 and MINS (MINS release v-0.1.01)

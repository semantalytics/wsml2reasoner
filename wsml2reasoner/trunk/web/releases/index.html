WSML2Reasoner Framework - Releases

Overview | Releases | Online Demo | Javadoc | View Source | Snapshot

The WSML2Reasoner Framework is packaged and distributed in several different variations in accord to reasoning capabilites and licensing issues. Each release provides a wsml2reasoner jar, wsml2reasoner-src zip file containing the source files, and wsml2reasoner zip file containing both source and binaries, as well as the WSMO and reasoning engine libraries for certain distributions.

A list of the most recent releases (v0.3), including source code, can be found here

  • WSML 2 Reasoner - WSMO Release - (LGPL) - wsml2reasoner-wsmo-v0_3

    • Includes necessary WSMO libraries. Reasoning libraries must be added.

  • WSML 2 Reasoner - Datalog Release - (GPL) - wsml2reasoner-datalog-v0_3

    • Includes WSMO libraries. Datalog reasoning libraries must be added. In addition to the IRIS and MINS reasoning engines (see below), wrappers are also currently implemented and tested for KAON2 (WSML-Flight).

  • WSML 2 Reasoner - Datalog Release (IRIS) - (GPL) - wsml2reasoner-datalog-iris-v0_3

    • Includes WSMO and IRIS (Datalog-based reasoning engine) libraries.

  • WSML 2 Reasoner - Datalog Release (MINS) - (GPL) - wsml2reasoner-datalog-mins-v0_3

    • Includes WSMO and MINS (Datalog-based reasoning engine) libraries.

  • WSML 2 Reasoner - DL Release - (LGPL) - wsml2reasoner-dl-v0_3

    • Includes WSMO libraries. DL reasoning libraries must be added.

  • WSML 2 Reasoner - DL Release (Pellet)- (LGPL) - wsml2reasoner-dl-pellet-v0_3

    • Includes WSMO, OWL, & Pellet (DL reasoning) libraries.

Which release is right for me?
The available releases are in accord to WSML variants (please refer to the WSML specification for futher description).
    • The WSMO Release can be built to handle reasoning with WSML-Core, -Flight, -DL, and -Rule ontologies, depending on the user's requirements. It is recommended that the non-expert user avoid using these releases and opt for either the Datalog or DL release.

      The Datalog Release is specifically designed to cater to reasoning with WSML-Rule ontologies; however this includes satisfying the reasoning requirements of the underlying WSML-Core and -Flight variants. The WSML-Rule variant is based on the Horn subset of F-Logic with negation under the well-founded semantics. The interested or more technical users are referred to the current WSML specification.

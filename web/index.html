WSML 2 Reasoner Framework

The WSML 2 Reasoner framework can translate ontology descriptions in WSML to predicates and rules. Additionally it provides a facade for easy integration of different reasoners.

Source and binary distributions can be obtained from the Snapshot directory (generated daily). Please note that this is a work in progress and should at best be considered alpha quality code. We are continually trying to improve the quality of the software. Comments and suggestions are welcome.

Demonstration

A simple Online Demo has been developed based up on this framework.

Dependencies

This software depends on WSMO4j (latest Snapshot)
For using the DL support you will need additionally the OWL-API

This framework handles translations only; to actually perform reasoning you will need a reasoning engine. Wrappers are currently implemented and tested for KAON2 (WSML Flight), MINS (WSML Rule), and Pellet (WSML DL).
